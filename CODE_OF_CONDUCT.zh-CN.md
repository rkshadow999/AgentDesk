# 贡献者公约行为准则

[English](CODE_OF_CONDUCT.md) | [简体中文](CODE_OF_CONDUCT.zh-CN.md)

## 我们的承诺

作为社区成员、贡献者和领导者，我们承诺为每个人提供免受骚扰的社区参与体验，不论其年龄、体型、可见或不可见的残疾、族裔、性征、性别认同与性别表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教、性身份或性取向如何。

我们承诺以有助于建设开放、友好、多元、包容和健康社区的方式行动与互动。

## 我们的标准

有助于为社区创造积极环境的行为包括：

- 对他人表现出同理心与善意。
- 尊重不同的意见、观点和经历。
- 提出并从容接受建设性反馈。
- 对自己的错误承担责任，向受影响的人道歉，并从经历中学习。
- 不仅关注个人利益，也关注整个社区的最佳利益。

不可接受的行为包括：

- 使用带有性意味的语言或图像，以及任何形式的性关注或挑逗。
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击。
- 公开或私下骚扰。
- 未经明确许可，公开他人的私人信息，例如实际地址或电子邮件地址。
- 其他在专业环境中可被合理认定为不当的行为。

## 执行责任

社区领导者负责解释并执行可接受行为标准，并会针对其认定为不当、威胁性、冒犯性或有害的行为，采取适当且公平的纠正措施。

社区领导者有权且有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，并会在适当时说明管理决定的理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中正式代表社区的情形。代表社区的例子包括使用官方电子邮件地址、通过官方社交媒体账号发布内容，或作为指定代表参加线上或线下活动。

## 执行

滥用、骚扰或其他不可接受的行为可以通过 GitHub 的内容举报与仓库管理工具报告。包含敏感个人信息或会造成安全风险的报告，可通过以下渠道私下提交：

https://github.com/rkshadow999/AgentDesk/security/advisories/new

所有投诉都会得到及时、公平的审查和调查。所有社区领导者都有义务保护事件报告者的隐私与安全。行为准则由独立的 AgentDesk 社区执行；请勿向上游公司联系人发送行为问题报告。

## 执行指引

社区领导者将按照以下社区影响指引，决定对任何其认定违反本行为准则的行为采取何种后果：

### 1. 纠正

**社区影响：** 使用不当语言，或出现其他被认定为不专业或不受社区欢迎的行为。

**处理后果：** 社区领导者发出私下书面警告，明确违规行为的性质，并解释该行为不当的原因。可能会要求公开道歉。

### 2. 警告

**社区影响：** 通过单一事件或一系列行为违反准则。

**处理后果：** 发出警告，并说明继续该行为的后果。在指定时间内不得与相关人员互动，包括主动联系执行行为准则的人员。这既包括社区空间内的互动，也包括社交媒体等外部渠道。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响：** 严重违反社区标准，包括持续的不当行为。

**处理后果：** 在指定时间内，临时禁止与社区进行任何形式的互动或公开交流。在此期间，不得与相关人员进行公开或私下互动，包括主动联系执行行为准则的人员。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响：** 表现出持续违反社区标准的行为模式，包括持续的不当行为、骚扰个人，或对某类人群进行攻击或贬损。

**处理后果：** 永久禁止在社区内进行任何形式的公开互动。
