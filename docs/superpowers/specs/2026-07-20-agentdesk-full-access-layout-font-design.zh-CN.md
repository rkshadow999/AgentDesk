# AgentDesk 完全访问、可调检查器与字体缩放设计

## 目标

为 Windows 客户端增加三个可持久化能力：一次确认后自动处理引擎工具审批的“完全访问”、可拖拽并记忆宽度的右侧检查器，以及同时作用于工作台、终端和 Diff 的字体缩放。

## 安全与所有权

- “完全访问”默认关闭，只能在原生 WinUI `ContentDialog` 中确认后开启。WebView 只能请求变更，不能自行越过原生确认。
- 开启后，Host 对当前或后续会话的 ACP 工具审批优先选择 `AllowOnce`。不向引擎写入永久规则，因此关闭设置后立即恢复逐次审批。
- 插件/MCP 安装、Windows UI Automation、凭据输入、备份恢复和明文 HTTP 等独立安全边界不受该开关影响。
- 审批没有 `AllowOnce`、引擎拒绝响应、会话已切换或出现异常时，Host 回退到普通审批窗口；这些情况不得发布全局“引擎失败”。
- 设置保存于 `%LOCALAPPDATA%\AgentDesk\ui-settings.json`，旧 schema 自动迁移，默认关闭。

## 检查器布局

- WinUI 主 Grid 使用 `* / 8 / Inspector` 三列，中间由原生 `Thumb` 提供鼠标、触摸和键盘调整。
- 检查器默认 360 DIP、最小 320 DIP、最大 960 DIP，同时至少为工作台保留 560 DIP。窗口缩小时只钳制当前显示宽度，重新放大后恢复用户的首选宽度。
- 左方向键增宽、右方向键缩窄，Shift 加速；Enter、Space 或双击恢复 360 DIP。模态窗口打开时分隔条不可交互。
- 宽度独立保存到 `%LOCALAPPDATA%\AgentDesk\window-layout.json`。缺失、损坏或写入失败静默回退，不能污染引擎状态。

## 字体缩放

- 设置提供 90%、100%、110%、125%、140% 五档，新安装和旧设置迁移默认 110%。
- CSS 字号由 10px 基准 rem 化，两个 WebView 收到偏好事件后立即更新根字号。
- Monaco Diff 与 xterm.js 使用同一个百分比更新编辑器字体并触发布局/fit，避免 canvas 字体不随 DOM 缩放。
- 只缩放文字，不使用视口宽度驱动字号；固定控件保留稳定尺寸并验证中文、键盘、高 DPI 和窄窗口无截断。

## 协议与数据流

`UiPreferences` schema 升至 4，增加 `fullAccessEnabled` 和 `fontScalePercent`。C# 与 TypeScript 严格协议同步增加同名字段。工作台 250ms 防抖保存完整偏好；Host 规范化、原生确认并持久化后广播权威快照，工作台与检查器只投影该快照。

## 验收

- 完全访问关闭时仍出现审批；开启并确认后普通 `execute` 自动继续；关闭后下一次重新弹窗。
- 无允许选项或自动响应失败时仍能手动选择，右上角不出现“引擎失败”。
- 检查器可拖到更宽、重启保持、窗口缩放后恢复首选宽度，键盘和双击可操作。
- 五档字体即时作用于聊天、输入框、设置、侧栏、终端和 Diff，重启保持。
- Web、C#、XAML policy、真实 sidecar、真实 Provider 和打包后的 EXE 验证全部通过。
